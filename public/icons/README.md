# Required PWA Icons

Please add the following icon files to this directory:

## Required Icons:
- `icon-192x192.png` - 192x192px icon for PWA
- `icon-512x512.png` - 512x512px icon for PWA  
- `apple-touch-icon.png` - 180x180px icon for iOS
- `favicon.ico` - 32x32px favicon for browsers

## Icon Requirements:
- Use your SmartCampus/SCL logo
- PNG format for app icons
- ICO format for favicon
- Square dimensions with proper padding
- Clean, scalable design

## Quick Generation:
You can use tools like:
- [Favicon.io](https://favicon.io) - Generate all formats from one image
- [PWA Builder](https://www.pwabuilder.com) - Generate PWA icons
- [Canva](https://canva.com) - Design and resize icons
